# Normal

**It shouldn't cost anything to use your phone less.**

Normal is a complete screen time solution — built to actually help you use your phone less. It's also free, open-source, and privacy-focused. No data collected. No data sold.

---

### Why Normal?

- **Unbypassable** — Set it up so you can't cheat your own rules.
- **Hardware blocker** — Any NFC device or QR code works as your physical key.
- **Dumb phone mode** — Block all smart apps and strip your phone back to basics.
- **Feature phone mode** — Block only the apps that waste your time.
